# kuyalloyd To-Do App

A beautiful, functional to-do list application built with HTML, CSS, and JavaScript.

## Features

✓ **Add new tasks** - Add tasks with title and optional description
✓ **Mark tasks as complete** - Click the Complete button to mark tasks done
✓ **Delete tasks** - Remove tasks you no longer need
✓ **Progress tracking** - See visual progress bar and completion percentage
✓ **Data persistence** - All tasks are saved in browser storage
✓ **Responsive design** - Works perfectly on desktop and mobile

## Technologies Used

- **HTML** - Structure and layout
- **CSS** - Styling and animations
- **JavaScript** - Interactivity and task management

## How to Use

1. Enter a task title in the input field
2. (Optional) Add a description in the textarea
3. Click "Add Task" or press Enter
4. Click "Complete" to mark a task done
5. Click "Delete" to remove a task
6. Watch your progress update automatically!

## Files

- `index.html` - Main HTML file
- `styles.css` - All styling (392 lines)
- `script.js` - JavaScript functionality (126 lines)

## Local Development

Simply open `index.html` in your web browser to get started!
